# 🚀 Advanced Task Manager (Django)

A simple yet powerful web-based Task Manager application built using **Django**. This project allows users to manage daily tasks efficiently with authentication support.

---

## 📌 Project Overview

This is a full-stack web application developed using Django (Python framework).
Users can securely log in and manage their tasks through a clean and user-friendly interface.

---

## 🔥 Features

* ✅ User Authentication (Login & Logout)
* ✅ Add New Tasks
* ✅ View Tasks on Dashboard
* ✅ Delete Tasks
* ✅ Simple & Clean UI

---

## 🛠️ Tech Stack

* **Backend:** Python (Django)
* **Frontend:** HTML, CSS
* **Database:** SQLite
* **Deployment:** Railway

## ⚙️ How to Run Locally

### 1️⃣ Clone the repository

```bash
git clone https://github.com/rakeshmalash22/task-manager-django.git
cd task-manager-django
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Apply migrations

```bash
python manage.py migrate
```

### 4️⃣ Run the server

```bash
python manage.py runserver
```

### 5️⃣ Open in browser

```
http://127.0.0.1:8000/
```

---

## 🔐 Admin Access

Create a superuser:

```bash
python manage.py createsuperuser
```

Login URL:

```
http://127.0.0.1:8000/login/
```
